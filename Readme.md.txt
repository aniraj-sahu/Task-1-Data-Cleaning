# Customer Personality Analysis - Data Cleaning

## Objective
The objective of this project is to clean and preprocess the Customer Personality Analysis dataset using Python and Pandas.

## Dataset
Customer Personality Analysis (Kaggle)

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Data Cleaning Steps
- Checked dataset structure
- Identified missing values
- Removed duplicate records
- Standardized column names
- Converted date column to datetime
- Filled missing income values using median
- Verified data types
- Created additional analytical features

## Output
- Cleaned dataset: marketing_campaign_cleaned.csv
